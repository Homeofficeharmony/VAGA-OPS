**Internal helper:**
```javascript
const STATE_COLOR = { frozen: ACCENT_HEX.red, anxious: ACCENT_HEX.amber, flow: ACCENT_HEX.green }

function getDateRange(windowDays) {
  return Array.from({ length: windowDays }, (_, i) => {
    const d = new Date()
    d.setDate(d.getDate() - (windowDays - 1 - i))
    return d.toISOString().slice(0, 10)
  })
}
```

**Export 1: `getActivationComparison(session)`** (for DVIZ-01 before/after comparison)
- Input: single session object with `activationBefore`, `activationAfter`, `state`
- If either activation value is null, return `[]` (no data to compare)
- Return: array of 2 data points:
  ```javascript
  [
    { x: 'Before', y: session.activationBefore, label: `${session.activationBefore}/10`, color: STATE_COLOR[session.state] || ACCENT_HEX.green },
    { x: 'After',  y: session.activationAfter,  label: `${session.activationAfter}/10`,  color: STATE_COLOR[session.state] || ACCENT_HEX.green },
  ]
  ```

**Export 2: `getShiftTrajectory(sessions, window = 7)`** (for DVIZ-02 shift-over-time trend)
- Input: sessions array, window (7 or 30 days)
- Generate date range for the window using `getDateRange(window)`
- For each date, find all sessions with recorded `shift` values, compute average shift for that day
- Return: array of points, one per day:
  ```javascript
  { x: 'YYYY-MM-DD', y: avgShift (or null if no sessions), label: avgShift !== null ? avgShift.toFixed(1) : '', color: '' }
  ```
  Per research recommendation: return `y: null` for days with no data (Phase 5 decides how to render gaps). The trailing point (today) may be a partial day — this is acceptable behavior, matching the VagusLogSidebar pattern.

**Export 3: `getDailyStateSeries(sessions, window = 7)`** (supporting: state-segmented counts)
- Input: sessions array, window (7 or 30 days)
- Generate date range for the window
- For each date, count sessions by state ('frozen', 'anxious', 'flow')
- Return: object with three arrays:
  ```javascript
  {
    frozen:  [{ x: 'YYYY-MM-DD', y: count, color: ACCENT_HEX.red }],
    anxious: [{ x: 'YYYY-MM-DD', y: count, color: ACCENT_HEX.amber }],
    flow:    [{ x: 'YYYY-MM-DD', y: count, color: ACCENT_HEX.green }],
  }
  ```
