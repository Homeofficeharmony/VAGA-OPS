   - Renders an SVG with `viewBox="0 0 140 90"` and `width="100%"` (responsive within the 300px panel).

   - The 22s auto-dismiss timer continues running -- if it fires during the result step, it calls `onRate` with the collected data including `activationAfter`.
   - Update the `collectedRef` to also track `activationAfter` so the timeout handler can include it.

5. Keep the existing `handleSkipActivation` behavior -- if user skips, no result step (activationAfter is null, so no chart).

**Important constraints:**
