  <action>
Create `ParticleField.jsx` — a canvas-based particle system driven by per-state config and breath phase.

**Props:** `selectedState` (string: 'frozen'|'anxious'|'flow'), `breathPhase` (string: 'inhale'|'hold'|'exhale', default 'inhale')

**PARTICLE_CONFIG object** (defined inside the file, keyed by state ID):
- `frozen`: count 28, color '#8ab4cc' (cool blue-grey), speedBase 0.18, speedVariance 0.08, sizeMin 1.5, sizeMax 3.5, opacityMin 0.06, opacityMax 0.22, directionSpread 0.4 (mostly vertical slow drift). breathSpeedMult: { inhale: 1.15, hold: 1.0, exhale: 0.55 }
- `anxious`: count 42, color '#c8a040' (warm amber), speedBase 0.55, speedVariance 0.35, sizeMin 1.0, sizeMax 2.8, opacityMin 0.08, opacityMax 0.30, directionSpread 1.8 (jittery scatter). breathSpeedMult: { inhale: 1.45, hold: 1.2, exhale: 0.70 }
- `flow`: count 32, color '#52b87e' (soft green), speedBase 0.28, speedVariance 0.12, sizeMin 1.8, sizeMax 4.0, opacityMin 0.05, opacityMax 0.18, directionSpread 0.0 (smooth orbital). breathSpeedMult: { inhale: 1.20, hold: 1.0, exhale: 0.80 }

**Mobile cap:** If `window.innerWidth < 600`, cap particle count at 30.

**Canvas setup (in useEffect, dependency: [selectedState]):**
1. Get canvas ref via `useRef`, get 2D context
2. DPR scaling: `canvas.width = rect.width * dpr; canvas.height = rect.height * dpr; ctx.scale(dpr, dpr)` where `dpr = window.devicePixelRatio || 1`
3. Initialize particles array in a `useRef` (NOT useState — never trigger re-renders from particle updates):
   - For frozen/anxious: random position, random velocity from speedBase +/- speedVariance, random size/opacity within min/max
   - For flow: orbital particles — each has cx, cy (scattered centers), radius (20-80), angle (random), angularVelocity (0.002-0.006, random direction)
4. Start rAF loop. On each frame:
   - `ctx.clearRect(0, 0, w, h)` (use CSS dimensions, not canvas pixel dimensions)
   - Compute `speedMult = config.breathSpeedMult[breathPhaseRef.current] ?? 1.0` (use a ref for breathPhase to avoid stale closure — update ref in a separate useEffect)
   - For frozen/anxious: update `p.x += p.vx * speedMult; p.y += p.vy * speedMult`, wrap edges
   - For anxious: add jitter — 4% chance per frame to kick velocity by random +/- 0.4, clamp speed to 1.2 max
   - For flow: `p.angle += p.angularVelocity * speedMult; p.x = p.cx + cos(angle) * radius; p.y = p.cy + sin(angle) * radius`
   - Draw each particle: `ctx.globalAlpha = p.opacity; ctx.fillStyle = config.color; ctx.beginPath(); ctx.arc(p.x, p.y, p.size, 0, PI*2); ctx.fill()`
5. Cleanup: `cancelAnimationFrame(rafId)` in useEffect return

**State-switch handling:** When `selectedState` changes, re-initialize particles (full reset). Do NOT try to lerp between states — the ImmersionBackground already provides visual continuity.

**Render:** A `<canvas>` element with `className="absolute inset-0 w-full h-full"` and `style={{ opacity: 0.85 }}`. No pointer events.

**Anti-patterns to avoid (from research):**
- Do NOT put particles in useState — use useRef
- Do NOT use CSS transitions on canvas — all animation in rAF loop
- Do NOT forget DPR scaling — causes blurry particles on retina displays
- Do NOT forget rAF cleanup on unmount — causes memory leaks
