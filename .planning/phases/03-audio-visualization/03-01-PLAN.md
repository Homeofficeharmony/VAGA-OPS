  <action>
Create `FrequencyBars.jsx` as a drop-in replacement for `LissajousVisualizer.jsx`. Follow the exact same canvas lifecycle pattern (canvasRef, rafRef, useEffect with rAF + cleanup).

Props: `{ playing, analyserRef, color }`

Canvas setup:
- `width={320} height={192} className="w-full h-48" style={{ display: 'block' }}`
- Matches LissajousVisualizer's exact canvas dimensions and CSS classes

Bar configuration:
- 40 bars (within the 32-48 range specified by user)
- Each bar ~4-6px wide with 2px gaps, calculated from canvas width
- Vertical bars rising from bottom of canvas

Smoothed bar heights (stored in a Float32Array ref, size 40):
- Pre-allocate `Uint8Array(1024)` ONCE in the useEffect (outside the draw function) for FFT data — do NOT allocate inside the rAF loop (GC pressure pitfall)
- Note: allocate the Uint8Array only when `analyserRef.current` is available; if null at effect start, allocate lazily on first non-null frame or use a ref to hold it

Live data path (when `playing && analyserRef.current`):
- Call `analyserRef.current.getByteFrequencyData(dataArray)`
- Sample lower 60% of bins (indices 0..~614) — binaural tones + pink noise live here
- Map 614 bins to 40 bars by averaging bin ranges
- Normalize: `(avg / 255) * canvasHeight` = target height
- Exponential smoothing with asymmetric attack/decay:
  - Rising (target > current): `current = current * 0.4 + target * 0.6` (fast attack)
  - Falling (target < current): `current = current * 0.88` (slow decay, organic feel)

Static fallback path (when `!playing || !analyserRef.current`):
- Use a pre-seeded array of 16 non-uniform heights like `[0.12, 0.18, 0.25, 0.20, 0.32, 0.15, 0.22, 0.28, 0.18, 0.12, 0.24, 0.30, 0.16, 0.20, 0.14, 0.26]`
- Scale by `canvasHeight * 0.35` for low-opacity decorative feel
- Smooth transition: `current = current * 0.94 + target * 0.06` (slow drift)

Bar rendering:
- Each bar gets a linear gradient from top to bottom: `color + 'ff'` at peak (full opacity) to `color + '44'` at base (dimmer)
- Minimum bar height of 2px to avoid invisible bars
- Clear canvas each frame with `ctx.clearRect(0, 0, W, H)`

The `color` prop is a hex string like `#52b87e`. Append alpha hex directly (e.g., `color + 'ff'`).

useEffect dependencies: `[playing, analyserRef, color]` — re-run when these change (matches LissajousVisualizer pattern).
