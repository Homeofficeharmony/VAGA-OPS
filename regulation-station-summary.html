🧠 01 Overview

Regulation Station is a calm-tech single-page application built on Polyvagal Theory. It helps work-from-home solopreneurs identify their current nervous system state (Frozen, Anxious, or Flow), then apply targeted somatic protocols, curated task lists, and binaural audio to shift toward regulated, productive work.

The app is fully offline-capable using localStorage. An optional Supabase backend enables cross-device session syncing, authentication, and team collaboration features.

🗂 03 Architecture

App Flow

  1. Init — ThemeProvider wraps app; check vagaFirstVisitComplete in localStorage; load sessions; calculate streak.
  2. First Visit — Show FirstVisitExperience (30-sec breathing timer + state intro). On complete, set localStorage flag.
  3. Main Layout — Header (fixed) → StateAssist → StateSelector → DailySummary → HRV Indicator → (if state selected) Dashboard Grid → TeamPanel → WeeklyIntelligenceCard → Footer.
  4. Overlays — Fixed-position layers rendered at bottom of App.jsx at various z-index levels: NeuralBackground (z-0), ImmersionBackground (z-1), content (z-10), modals (z-50 to z-100).
  5. State Transitions — Selecting a state resets timer; switching states clears all transient overlays and task load; completing any reset triggers PostResetCheckin.

State Management

No external state library. All central state lives in App.jsx and is passed down as props:

selectedState'frozen' | 'anxious' | 'flow' | null — drives entire UI
panicOpenboolean — shows PanicReset overlay
focusOpenboolean — shows FocusMode overlay
flowLockOpenboolean — shows FlowLock overlay
ruptureOpenboolean — shows RuptureModal
isImmersiveboolean — toggles immersion mode
checkinPending{ source, accentHex, state } | null — triggers PostResetCheckin

Data Flow

stateData.js (STATES)
  └─► App.jsx (selectedState → stateData = STATES[selectedState])
        ├─► StateSelector (read + set selectedState)
        ├─► StealthReset (stateData.reset)
        ├─► TaskFilter (stateData.tasks)
        ├─► AudioPlayer + useAudioEngine (stateData.audio)
        └─► All overlays (accentHex, selectedState)

🪝 05 Custom Hooks

All hooks live in src/hooks/. They handle side effects, Web Audio, session persistence, and analytics — keeping components purely presentational.

useSessionLog()
Persists sessions to localStorage key vaga-sessions (max 90 entries). Syncs to Supabase nervous_system_sessions table when authenticated. Returns { sessions, logSession, clearSessions }. Session shape includes state, type, durationSec, resetCompleted, outcome, shift, flowMinutes, activationBefore, activationAfter.
useAudioEngine({ carrierHz, beatHz })
Web Audio API binaural beat engine. Left ear = carrierHz, right ear = carrierHz + beatHz. Also generates pink noise (Paul Kellett algorithm, low-pass at 4kHz). Handles browser autoplay policy. Returns { playing, play, pause, volume, setVolume, supported }.
useStreak(sessions)
Memoized streak calculation from session array. Extracts unique calendar days, sorts descending, counts consecutive days from most recent. Returns { streak, lastCheckin }.
useTeam()
Team CRUD via Supabase RLS. createTeam(name), joinTeam(inviteCode), leaveTeam(), fetchMembers(teamId). Returns { team, members, createTeam, joinTeam, leaveTeam }. Graceful no-op if Supabase is null.
useSessionMetrics(sessions) — pure exports
computeMetrics() returns averageActivation, averageDelta, stateFrequency. generateWeeklySummary() returns dominantState, avgRecoveryTime, mostEffectiveProtocol, volatilityScore (proportion of state transitions across consecutive sessions).
useBreathTimer(running)
Cyclic breathing phase manager for animations. 13-second cycle: 4s inhale, 1s hold, 8s exhale. Uses requestAnimationFrame. Returns { phase, cycleProgress, cycleTimeMs }.
useCompletionTone()
Plays a short Web Audio API completion beep when protocols finish.
useHRV()
HRV proxy calculation for the gauge indicator in the Header.

✨ 07 Key Features

⚡ Panic Reset
30-second emergency regulation triggered by the bottom-left ⚡ button or Cmd+Shift+R. Expanding/contracting SVG breathing orb synced to inhale/exhale cues. Auto-proceeds to PostResetCheckin on completion.
🔒 Flow Lock
90-minute deep-work blocker overlay for the Flow state (Cmd+F). Minimal distraction UI, tracks elapsed minutes, plays completion tone, logs flowMinutes to session log on exit.
🌿 Immersion Mode
Toggled via I key. Shows procedural neural background, breathing orb animation, vagus log sidebar, ambient soundscape. Hides all operational noise for distraction-free deep work.
🎧 Binaural Audio
Real Web Audio API binaural beats generated on-device. Frozen: Theta 5 Hz (180+5 Hz). Anxious: Alpha 10 Hz (200+10 Hz). Flow: Gamma 40 Hz (200+40 Hz). Pink noise backing. Lissajous canvas visualizer.
📊 Session Analytics
Daily Summary strip, 7-day consistency graph, Weekly Intelligence card with dominant state, average recovery time, most effective protocol, and volatility score. Streak tracking with milestone celebrations.
🧑‍🤝‍🧑 Team Features
Create/join teams with 6-char invite codes. View roster with members' streaks and latest states. Own session data only visible to self (Supabase RLS). Graceful no-op when offline.
📱 PWA / Offline-First
Fully functional without Supabase using localStorage. PWA-enabled via vite-plugin-pwa with auto-update and Google Fonts cache-first. Installable as standalone app on desktop/mobile.
🎨 3-Theme System
Dark (default), Light, and Pastel themes via CSS custom properties on data-theme. Biophilic, non-neon accents: sage green, amber gold, terracotta. Persisted in localStorage.

Design Principles

  • Biophilic, not neon — warm earth tones, no harsh bright colors
  • Breathe animations — 4-second cycles that mirror physiology
  • Monospace for data — JetBrains Mono at small sizes for precision
  • Soft glows — state-colored radial shadows, no sharp UI borders
  • Dark default — optimized for WFH comfort (#0f1410 background)
  • Large touch targets — buttons minimum 2.5rem for accessibility
